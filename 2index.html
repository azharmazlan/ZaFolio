Cara mendapatkan Telegram ID:
  1. Klik pautan https://t.me/UserInfoBot untuk dapatkan Telegram ID anda.
  2. Tekan START pada bot tersebut.
  3. Bot akan memaparkan nombor Telegram ID anda.
  4. Salin nombor Telegram ID tersebut.
  5. Kemudian klik pautan https://t.me/Alamanda_VocKAF_bot.
  6. Hantar nombor Telegram ID yang diperoleh tadi kepada bot Rasmi VocKAF tersebut supaya akaun anda didaftarkan untuk menerima notifikasi akan datang.
  7. Selepas mesej berjaya dihantar, kembali ke sini dan masukkan nombor Telegram ID tersebut di ruangan 'ID Telegram''.
